# ReUse | Mobile

## Contexto da Sprint
Esta é a **Sprint 1** do projeto **ReUse**, uma plataforma digital voltada à economia sustentável, reutilização de produtos e consumo consciente.  
O objetivo inicial é criar o **primeiro contato do usuário com o aplicativo mobile**, por meio de uma interface simples, clara e bem estruturada.

---

## Objetivo da Sprint
- Construir a base do aplicativo mobile ReUse.  
- Organizar a interface utilizando **React Native**.  
- Apresentar de forma clara a proposta do produto.  
- Criar componentes reutilizáveis para garantir escalabilidade futura.  

---

## Decisões de Interface
- **Tela inicial (Home):** apresenta o nome e propósito da plataforma, com chamada para conhecer mais sobre o projeto.  
- **Tela "Quem Somos" (About):** explica o problema abordado e a proposta de valor da ReUse.  
- **Componentes reutilizáveis:** criados para botões (`CustomButton`, `CustomOutline`) e backgrounds (`Background`, `BgAbout`).  
- **Organização visual:** uso de `Flexbox`, cores contrastantes e sobreposição de imagens para dar clareza e destaque.  
- **Navegação:** implementada com **React Navigation (Stack)**, embora opcional nesta fase.  

---

## Telas Desenvolvidas

### Home
- Logo da ReUse.  
- Texto explicativo: *"Troque objetos, reduza desperdício e consuma com consciência."*  
- Botões: **Sobre a ReUse!** e **Login**.  
- Link para cadastro.  

### About
- Título: **Quem Somos**.  
- Texto explicativo sobre a iniciativa ReUse.  
- Botões: **Voltar** e **Login**.  

### LoadingScreen
- Logo de carregamento.  
- Indicador de progresso (`ActivityIndicator`).
